## River Raider 1863

### Purpose
This repository contains a prototype for an educational game about Harriet Tubman and the Combahee River Raid, created for DGMD E-57 (Section 1), Harvard Extension School, Spring 2026.

### Getting Started

1. Clone this repository to your local machine.
2. In the project directory, install dependencies:

   ```bash
   npm install
   ```
3. To start development server:
   ```bash
   npm run dev
   ```
4. Open your browser to the URL shown in the terminal (by default, http://localhost:5173).


---

### Sources

The framing and interpretation in *River Raider 1863* draw on the following works:

- **Fields-Black, Edda L.**  
  *Combee: Harriet Tubman, the Combahee River Raid, and Black Freedom during the Civil War*.  
  Oxford University Press, 2023.

> Note: In this prototype, some documents are adapted or composited for teaching purposes. Captions and summaries are written for student audiences and are not quotes from the original records.
